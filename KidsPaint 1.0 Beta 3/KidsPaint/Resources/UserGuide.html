User Guide

KidsPaint
by Fivethirty Softworks

KidsPaint is a simple, open-source painting app for kids (and parents!) that focuses on creativity without clutter. There are no documents to manage and no confusing modes — just open the app, paint, and save.

KidsPaint runs on macOS and iPadOS, with a consistent, kid-friendly interface across all platforms.

What You Can Do

  • Draw freely using multiple brush styles.
  • Add large, fun stickers that can be moved, resized, rotated, recolored, and deleted.
  • Change colors instantly using big, easy-to-tap color buttons or the full color picker.
  • Import photos and draw directly on top of them.
  • Use touch, mouse, trackpad, Apple Pencil, or a game controller.
  • Save artwork as PNG — no special file formats.

Main Tools

Brush

  • Freehand drawing tool.
  • Adjust size with the slider.
  • Choose different brush tips.

Eraser

  • Erases brush strokes.
  • Size adjustable.

Stickers

  • Tap to place large stickers.
  • Stickers are intentionally oversized and easy to grab.

Move

  • Pan the canvas.
  • Zoom in and out.

Bottom Control Bar

Color Selection

  • Large preset color buttons.
  • Full color picker for custom colors.
  • Color applies to new strokes and new stickers.

Size Control

  • Brush / Eraser: line thickness.
  • Stickers: default sticker size.

Brush Tips

  • Round
  • Square
  • Chisel
  • Crayon
  • Spray
  • Neon

Working With Stickers

  • Tap to select a sticker.
  • Drag to move.
  • Resize and rotate using on-screen handles.
  • Delete using the Delete key or context action.

Game Controller Support

Supported on macOS and iPadOS using any controller recognized by the system.

  • Left stick – Move cursor
  • Primary button – Draw / place
  • D-Pad – Change colors and brush tips
  • Triggers – Adjust size or zoom (when available)
Tip: Click or tap the canvas once to ensure it has focus before using a controller.

Background Images

  • Import PNG or JPG images.
  • Draw and place stickers on top.
  • Remove the background image at any time.

Saving Artwork

  • Saved as standard PNG files.
  • Default filename includes date and time.
  • No documents or project files to manage.

Clearing the Canvas

  • Tap the Trash icon.
  • Confirm to prevent accidents.

View Controls

  • Reset View returns zoom and position to default.
  • Pinch or scroll to zoom.
  • Use Move tool to pan.

Legal & Beta Notice

KidsPaint is beta software. Features may change and bugs may exist. Use at your own risk.
